# Singularity Fizkin

A Singularity image for Fizkin.

# To Build

Use the "img" target of the Makefile:

    $ make img

# To Run

The Singularity image can be run like an executable:

    $ ./fizkin.img

Or you can use Singularity:

    $ singularity run fizkin.img
